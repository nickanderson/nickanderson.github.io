Installing NRPE on XenServer

Mar 4, 2010

I like to have as little run in dom0 as possible. However some things you really need checked from dom0, like the status of your raid perhaps. Just some quick instructions on getting Nagios NRPE running in XenServer.

  1. Install EPEL repository and disable it by default (remember we donâ€™t want to accidentally install unnecessary packages) wget http://download.fedora.redhat.com/pub/epel/5/$(uname -i)/epel-release-5-3.noarch.rpm rpm -hiv epel-release*.rpm sed -i ’s/enabled=1/enabled=0/g' /etc/yum.repos.d/epel.repo

  2. Install nrpe and configure it to start on boot yum install –enablerepo=epel nrpe chkconfig nrpe on

  3. Modify the firewall to allow NRPE connections. Add the following before the REJECT line in /etc/sysconfig/iptables -A RH-Firewall-1-INPUT -m state –state NEW -m tcp -p tcp –dport 5666 -j ACCEPT

  4. Restart your firewall and start nrpe restart your firewall , and start nrpe /etc/init.d/iptables restart && /etc/init.d/nrpe start

  5. Configure nrpe like normal and have fun
