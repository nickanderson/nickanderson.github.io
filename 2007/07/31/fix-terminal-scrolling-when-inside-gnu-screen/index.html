Fix terminal scrolling when inside GNU Screen

Jul 31, 2007

My post on Automatic session logging and monitoring with GNU screen seems to have been well received. I have had more hits by far on that post today than ever before. I noticed a nagging issue with screen and a friend of mine Travis Cline pointed me to the fix.

Isnt it annoying when your in screen and cant use the scroll wheel to scroll up in your buffer? The fix does not work well when you are using multiple windows inside a single screen especially when using split mode but for the times you use one window per screen its pretty handy.

Add the following to your .screenrc.
